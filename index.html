    Erupt

    A Crystal logging library that comes with 14 different log types.

    Installation

    1. Add the dependency to your shard.yml:
    dependencies:
  erupt:
    github: greatgodapollo/erupt
    1. Run shards install

    Usage

    require "erupt"

    Check out the examples in /examples/
